Um Guia sobre Web Scraping com NodeJS e Typescript: Parte 1 — Introdução

Bem vindos ao primeiro artigo sobre Web Scraping com NodeJS e Typescript, neste artigos iremos abordar sobre o conceito de Web Scraping.

Antes de tudo, deixarei claro que o intuito desse guia é ensinar a fazer Web Scraping ou raspagem de dados de websites, entretanto não recomendo a prática do mesmo em websites, se possível peça ao website um acesso a sua API.

O que é Web Scraping?

Web Scraping é uma técnica de programação que permite obter dados e informações de um determinado website, com o uso de web scraping podemos gerar históricos de preços, baixar musicas e videos, e criar inteligências artificiais gerativas como o MidJourney e o DALL-E.

Web Scraping vs Web Crawling

Um detalhe importante sobre web scraping é sua variação chamada de Web Crawling, muito pensam que ambos significam a mesma coisa, só que nem tanto. Web Scraping extrai dados de um único website ou página, enquanto o Web Crawling procura por outras páginas e websites.

Linguagens para Web Scraping

Praticamente qualquer linguagem de programação consegue fazer web scraping, até mesmo Bash, só que algumas linguagens possuem mais facilidades com certos tipos de dados como JSON, essas linguagens possuem menos tipagens e são mais dinâmicas, como Python, Javascript, Go e PHP.

Web Scraping e Headless Browsers

Muitos websites não entregam uma página completa para o usuário, o restante da página é carregado dinamicamente através de requisições AJAX, que usam Javascript, com isso surgiu a ideia de simular um navegador aberto e automatizado, e daí vem o termo Headless Browser, ou navegador sem cabeça. Headless Browsers são navegadores sem um interface gráfica que permite a simulação de ações feita por um usuário, a vantagens não fica só no carregamento de elementos dinâmicos, os headless browsers utilizam um navegador de verdade, com isso fica difícil de ser detectado como um bot.
